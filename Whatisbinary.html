What is Binary?

Binary


Binary is a Base-2 number system meaning that it only uses 0 and 1 (the normal number system people use is Base-10 since it uses 0-9). Each one or zero is known as a bit and eight bits together make a byte. In binary each one equates to the value of the spot it is in for example for right to left the first 8 digits values go 1,2,4,8,16,32,64,128 and then this exponential pattern. The easiest way to convert binary to decimal is by adding all the values where there is a one together for example 10110111 would be 1+2+4+16+32+128 which equals 183.


Binary is most known for being used in all computers (except quantum computers). Binary is used to store the data of computers and then converts it into characters if a human needs to read the data. Binary is also used to show the maximum storage of a computer. This is mostly shown in the way of gigabytes (eight billion bits each) or terabytes (8 trillion bits each).
